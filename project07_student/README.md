# SoundDino
用自己的聲音遊玩Chrome Dino <br>

## 建立虛擬環境
```shell
python -m venv SoundDino
```
<br>

## 安裝所需套件
```shell
pip install sounddevice
pip install scipy  
pip install keyboard  
pip install librosa  
pip install scikit-learn  
pip install resampy
```
<br>

## 執行主程式
```shell
python main.py
```
<br>
